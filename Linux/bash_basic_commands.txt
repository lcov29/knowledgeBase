whatis <command>		short description of a command
man <command>			detailed description of a command
apropos	<keyword>		search for a command			
alias				list or set alias for command
history				list contents of .history_bash (history of all commands)
history -c			delete bash history
!<Number>			executes command with <Number> in bash history
!!				execute last command again
CTRL+A				move cursor to the start of the current input
CTRL+E				move cursor to the end of the current input
ALT+B				move cursor one word backward
ALT+F				move cursor one word forward
CTRL+T				switch last two characters after the cursor
ALT+T				switch last two words after the cursor
